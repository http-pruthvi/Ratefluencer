## 📽️ Slide 5: Machine Learning Core
*   **Headline:** Behind the Mathematics of Influence
*   **Bullet Points:**
    *   **Feature Engineering:** Extracts 9 high-impact features (Engagement rates, posting frequencies, growth volatility, SBERT dense vectors, and follower ratios).
    *   **Unsupervised Bot Auditing:** Using Isolation Forest to isolate anomalies closer to the root of randomized decision trees (short path lengths $h(x)$).
    *   **Additive Sequence Modeling:** FB Prophet capturing organic follower trajectories:
        $$y(t) = g(t) + s(t) + h(t) + \epsilon_t$$
    *   **Gradient Boosting Trees:** Iteratively fits weak estimators to minimize localized regression objectives, preventing creator classification overfitting.
*   **Suggested Visual:** A flow diagram tracing how raw creator metrics flow into a Feature Engineering block, split into unsupervised bot auditing and time-series growth nodes, and finally merge into a Gradient Boosting classifier to generate the proprietary **Ratefluencer Score™**.
